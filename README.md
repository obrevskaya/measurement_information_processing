# Методы обработки измерительной информации

## Описание проекта

Программа реализует основные методы математической статистики и теории случайных процессов, позволяя проводить анализ экспериментальных данных, проверять статистические гипотезы и исследовать свойства случайных процессов.

## Реализованные возможности

### Поиск промахов в результатах измерений

Реализован алгоритм обнаружения грубых ошибок измерений (промахов) на основе статистического анализа выборки.

Функциональность включает:

* вычисление выборочного среднего значения;
* вычисление среднеквадратического отклонения;
* оценку вероятности ложного обнаружения;
* автоматическое выявление аномальных наблюдений;
* визуализацию обнаруженных промахов на графиках.

### Проверка гипотезы о нормальном распределении

Для анализа случайного процесса реализованы несколько статистических критериев проверки нормальности распределения.

#### Метод моментов

Проверка гипотезы о нормальном распределении по:

* выборочной асимметрии;
* выборочному эксцессу.

#### Критерий Пирсона (χ²)

Сравнение эмпирического распределения наблюдений с теоретическим нормальным распределением.

#### Критерий Колмогорова

Сравнение эмпирической и теоретической функций распределения случайной величины.

### Расчёт корреляционной функции

Реализован расчёт корреляционной функции случайного процесса для исследования статистических связей между отсчётами.

### Проверка стационарности случайного процесса

#### Проверка по дисперсии

Используется критерий Кохрана для анализа устойчивости дисперсии на различных участках наблюдений.

#### Проверка по математическому ожиданию

Используется критерий Фишера для проверки неизменности среднего значения случайного процесса.

### Проверка эргодичности

Реализована проверка гипотезы об эргодичности стационарного случайного процесса на основе нормированной корреляционной функции.

## Используемые технологии

* MATLAB

## Получаемые результаты

В процессе работы программа автоматически формирует:

* графики с выделенными промахами;
* гистограммы распределения вероятностей;
* графики эмпирических и теоретических функций распределения;
* корреляционные функции случайных процессов;
* графики проверки эргодичности.
